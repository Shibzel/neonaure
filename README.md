# Néonaure

Néonaure est version alternative du célèbre jeu Sudoku.

Codé par un groupe de 3 personnes à l'occasion d'une évaluation en IHM dans une formation de BUT informatique, avec Python et Qt.

## Setup et lancer le programme

Dans le dossier courant, préférablement dans une venv :

```sh
#python3 -m venv .venv      # À décommenter
#source .venv/bin/activate  # Sous linux, ou autre système supportant bash
#./.venv/bin/Activate.ps1   # Sous Windows, depuis Powershell
pip install -r requirements.txt
python3 main.py
```

> Assurez vous que Python est dans le path. Python `3.14` a été utilisé, mais le programme devrait aussi fonctionner pour les version >= `3.10`.
